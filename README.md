# HHS Unaccompanied Children Program Dashboard
# Overview
Streamlit-based analytics dashboard to visualize the UAC Program data from Health and Human Services (HHS). Tracks capacity, care load, and discharge trends for unaccompanied children.
## Key Metrics
- **Total Active Children:** 2502
- **In HHS Care Facilities:** 2484  
- **Discharged:** 18
# Tech Stack
- **Frontend:** Streamlit
- **Data Processing:** Pandas
- **Visualization:** Plotly
- **Language:** Python
# Features
- Date range filtering
- Real-time KPI cards
- Interactive time-series charts
- Discharge vs capacity trends

# Data Source
HHS Unaccompanied Alien Children Program Dataset
# How to Run
1. Clone repo
2. `pip install -r requirements.txt`
3. `streamlit run main.py`
